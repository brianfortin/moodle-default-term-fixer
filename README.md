# Moodle Default Term Fixer

Automatically selects the current term as the default term on the Moodle homepage.
For semesters beyond Fall 2025, the target value ID will likely have to be updated depending on the value assigned by Moodle.

## Installation
1. Install the [TamperMonkey] or [Greasemonkey] extension on Chrome.
2. Open the file in the extension and enable it. You must have developer mode enabled on Chrome for the extension.
